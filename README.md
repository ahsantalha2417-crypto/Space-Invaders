# Space-Invaders
This repository is a simple implmentation of the Space Invaders game. In this game, 25 spaceships will gradually advance down the screen towards the player, the player's objective is to eliminate all the enemy ships before all lives are exhausted. Player can move in all 4 directions using the arrow keys. The player can eliminate enemies by shooting lasers with the space key, every time an enemy is elimated, player gains 100 points. If an enemy reaches the player or the bottom of the screen, the enemy will dissapear and one of the player's lives will be lost. Game is over when either the player eliminates all enemies and wins, or loses all their lives and loses. Pressing the enter key will allow player to restart the game.
